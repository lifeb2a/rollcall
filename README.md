# 📋 多語點名系統 Rollcall for Activity

> 專為多國籍活動設計的點名工具，支援本人錄音、AI 辨識名單、離線點名播放。

---

## ✨ 功能特色

- 📷 **拍照辨識名單** — 拍一張名單照片，AI 自動辨識所有姓名（需網路，只做一次）
- 🎙 **本人錄音** — 每位參與者錄下自己念名字的聲音
- 🔊 **離線點名播放** — 點名時播出本人聲音，不需網路
- 📦 **匯出共享** — 錄音打包進 HTML，用 WhatsApp 傳給所有工作人員
- 🌐 **多語支援** — 中、英、日、韓、阿拉伯、泰、越南、馬來、法、德文
- 📱 **手機優先設計** — 介面專為手機操作設計

---

## 🚀 使用流程

### 第一階段：匯入名單（需網路）
```
打開工具網址
→ 設定頁籤
→ 點「📷 拍照辨識名單」上傳名單照片
→ AI 自動辨識，確認後載入
```

### 第二階段：錄音（不需網路）
```
錄音頁籤
→ 點任一姓名 → 全螢幕錄音畫面
→ 手機遞給對方
→ 對方按 🎙 念名字 → ⏹ 停止
→ ▶ 試聽確認
→ 全部錄完後按「📦 匯出含錄音的點名檔」
→ 用 WhatsApp 傳給所有工作人員
```

### 第三階段：點名（不需網路）
```
點名頁籤
→ 點 🔊 播出本人聲音
→ 本人舉手
→ 打勾，自動跳下一位
```
